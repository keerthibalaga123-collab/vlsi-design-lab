# Experiment 3 - 4-bit ALU

## Aim

To design a 4-bit Arithmetic Logic Unit (ALU) using Verilog HDL.

## Objective

The ALU performs different arithmetic and logical operations based on the selection input.

## Operations

| Select | Operation |
|---------|-----------|
|000|Addition|
|001|Subtraction|
|010|AND|
|011|OR|
|100|XOR|

## Inputs

- A[3:0]
- B[3:0]
- Sel[2:0]

## Output

- Result[3:0]

## Applications

- CPU
- Microprocessor
- Digital Signal Processing
- Embedded Systems

## Files

- verilog/alu4bit.v

## Result

Successfully implemented a 4-bit ALU using Verilog HDL.
