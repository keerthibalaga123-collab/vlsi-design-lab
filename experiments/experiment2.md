# Experiment 2 - Full Adder

## Aim
To design and implement a Full Adder using Verilog HDL.

## Theory
A Full Adder adds three one-bit binary inputs:
- A
- B
- Carry In (Cin)

Outputs:
- Sum
- Carry

### Boolean Equations

Sum = A ⊕ B ⊕ Cin

Carry = AB + BCin + ACin

## Applications

- Arithmetic Logic Unit (ALU)
- Binary Adders
- Processors
- Digital Systems

## Result

Full Adder implemented successfully using Verilog HDL.
